#  AutoDrive — Car Listing & Test Drive Booking Website

A comprehensive, multi-page car listing website with modern design, advanced features, and full client-side functionality.

---

##  Features

-  **User Authentication** — Register and login with localStorage persistence
-  **Browse & Filter Cars** — Filter by make, model, price, body type, fuel type
-  **Car Comparison** — Compare up to 3 vehicles side-by-side
-  **Loan Calculator** — Interactive sliders for real-time payment calculation
-  **Test Drive Booking** — Full booking system with form validation
-  **Dealership Locator** — View dealerships with ratings and contact info
-  **Save Favorites** — Save and manage preferred cars
-  **Blog & Articles** — Industry news and car reviews
-  **FAQ Section** — Interactive accordion Q&A
-  **Fully Responsive** — Works on desktop, tablet, and mobile
